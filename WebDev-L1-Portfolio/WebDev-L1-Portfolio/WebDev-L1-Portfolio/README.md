# Personal Portfolio Website

## Oasis Infobyte Web Development Internship

### Level 1 - Task 2

## Objective

The objective of this project is to create a responsive Personal Portfolio Website using HTML and CSS. The portfolio serves as a digital resume to showcase personal information, technical skills, projects, and contact details.

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)

## Features

- Responsive Navigation Bar
- Hero/Profile Section
- About Me Section
- Skills Section
- Projects Section
- Contact Section
- Smooth Scrolling Navigation
- Responsive Design
- Modern User Interface

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click *index.html* or open it using any web browser.
4. Explore the portfolio website.
