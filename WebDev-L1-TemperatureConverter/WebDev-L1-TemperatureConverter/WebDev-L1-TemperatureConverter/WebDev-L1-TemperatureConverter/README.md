# 🌡️ Temperature Converter Website

## 📌 Objective

The Temperature Converter Website is a simple and interactive web application that converts temperature values between *Celsius (°C), Fahrenheit (°F), and Kelvin (K)*. It provides accurate conversions with input validation and a user-friendly interface.

---

## 🚀 Features

- Enter a numeric temperature value
- Select the input unit (Celsius, Fahrenheit, or Kelvin)
- Convert temperature with a single click
- Displays converted values in all three units
- Input validation for non-numeric or empty values
- Prevents temperatures below absolute zero
- Clean and responsive user interface
- Works on desktop and mobile devices

## 🌡️ Temperature Conversion Formulas

### Celsius to Fahrenheit

F = (C × 9/5) + 32


### Celsius to Kelvin

K = C + 273.15


### Fahrenheit to Celsius

C = (F − 32) × 5/9


### Fahrenheit to Kelvin

K = (F − 32) × 5/9 + 273.15


### Kelvin to Celsius

C = K − 273.15


### Kelvin to Fahrenheit

F = (K − 273.15) × 9/5 + 32
